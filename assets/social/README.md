# Sigil Social Assets

These assets are designed for link previews and launch posts rather than app
icons or in-product branding.

| File | Size | Use |
| --- | --- | --- |
| `sigil-social-preview.svg` | 1280 × 640 | Editable source for the shared preview composition. |
| `sigil-social-preview.png` | 1280 × 640 | GitHub Social Preview and Open Graph/Twitter cards. |

The preview intentionally uses a solid dark background so it remains legible
on platforms that do not preserve transparency. Keep the PNG below 1 MB for
GitHub upload compatibility.
