# Sigil 45-second demo

This directory contains the public launch demo built from a real Sigil TUI run
in a disposable repository.

- `sigil-45-second-demo.mp4`: H.264, 1920 × 1080, 30 fps.
- `sigil-45-second-demo.webm`: VP9 fallback for the website.
- `sigil-45-second-demo-poster.png`: approval-review poster frame.
- `sigil-45-second-demo.en.vtt`: English captions.
- `sigil-45-second-demo.zh-CN.vtt`: Simplified Chinese captions.
- `sigil-45-second-demo-end-card.svg`: editable branded end-card source.

The canonical prompt, recording constraints, timing, and acceptance checklist
live in [`dev/docs/launch/45-second-demo-script.md`](../../dev/docs/launch/45-second-demo-script.md).
Raw recordings remain local and are not published.
